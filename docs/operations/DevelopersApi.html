This is the API

sunrise

By passing in the appropriate arguments, you can get the sunrise time at a specific location.

URL

https://virtserver.swaggerhub.com/sgv-deashboard/sgvd-sunset/1.0.0/sunrise

HTTP Method

GET

Response Type

sunTimeJson

Parameters

  • Query: lat Float

    The lattitude of the place you want the sunrise of.

  • Query: lon Float

    The longitude of the place you want the sunrise of.

  • Query: date date

    The date at which you want to calculate the sunrise
