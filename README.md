# sgvd-sunset

## Example usage

![Curl example](./docs/curl.svg)

```bash
$ curl "https://sgvd-sunset.herokuapp.com/sunrise?lat=50.93134&lon=5.40244&date=2021-11-23"
```

```json
{
  "datetime": "2021-11-23T16:41:20.202267+01:00",
  "location": {
    "latitude": 50.93134,
    "longitude": 5.40244
  }
}
```
